BOX MODEL....!

Every element that can be displayed on a web page is comprised of one on more rectangle boxes. CSS box model typically describes how these rectangle boxes are lied out on a web pages or website. these boxes can have different properties and can interact with each otherr in different ways, but every box has a contenyt area and optional surrounding padding, border, margin areas
